This repository stores the data of FoME.

1.*FoME/ExecutionData/*:
 
The representative execution traces (RTR) are in the file **$projectName$_execution_traces.txt**.

The tree visualization of RTR are in the file **$projectName$_execution_traces.tree**.


2.*FoME/monitorLogs/*:

**kieker-*.dat** files are monitored logs for each project.


3.*FoME/testcaseNames/*:

The designed testcase lists are in files **$projectname$_testcase_name.txt**.


4.*FoME/uncoveredTypes/*:

**class-distribution_detail.pdf** lists all the uncovered classes according to uncovered categories for each project.
