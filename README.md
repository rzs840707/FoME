This repository stores the data of FoME.

*FoME/$projectName$/ExecutionData*: 

The desgined testcase name lists are in the file **$projectName$_testcase_name.txt**.

The representative execution traces (RTR) are in the file **$projectName$_execution_traces.txt**.

The tree visualization of RTR are in the file **$projectName$_execution_traces.tree**.
